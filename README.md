## Boxsite
<br>

> [Desde el Ring](http://www.desdeelring.com) is a box website with news and info about the fighters

### Getting Started

  * TOOLS: node, yarn
  * clone the repository
  * `yarn install`
  * `yarn start` will start the app on port :3001
